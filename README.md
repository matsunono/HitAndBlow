1.ゲームのルール
　Hit＆Blowとは、コンピュータが設定した4桁の数字を当てるゲームです。
　設定した4桁の数字の中に同じ数字が二度含まれることはありません。
　Hitは数字とその位置が合っている状態のことを指し、
　Blowは数字は合っているが位置が合ってない状態のことを指します。
　4Hit出来たらゲームクリアです。

2.画面の説明
　①Startボタンを押すとランダムな4桁の数字が生成されます。
　②数字キーを押すか、数字ボタンを押すと数字が上部の欄に打ち込めます。
　③間違えた場合はClearボタンを押すと打ち直せます。
　④Checkボタンを押すと①で生成された数字と打ち込んだ数字を照らし合わせて、
　幾つがHitで幾つがBlowなのか返し、右上の欄に表示します。

　・照らし合わせた履歴は左の欄に表示されます。
　・下部にある小さい1～9のボタンはメモとして使ってください。
